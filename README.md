# Dataviz Components

This is a react library.

## Install

You can install with

```
npm i dataviz-components@latest
# or yarn
yarn add dataviz-components@latest
# or pnpm
pnpm add dataviz-components@latest
# or bun
bun add dataviz-components@latest
```
